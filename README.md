# DiscordBotofGreenpath
A Casual bot of discord that focuses on the enviroment.
It is a points based discord bot that uses quick challenges to the player related to
the climate change or enviromental issues, it helps users that are unaware of this planetary issue to
learn about it making it quick and interesting.

Commands
/profile
    "/Profile" is a quick and not so complex command that allows the user to see on which state of 
    the bot is in, shows levels, and points. Levels are obtained by doing some certain amount of points
    being 4 the maximun level with a total of 18 points.
/hi
    it shows a quick response in which uses a image and a short "about me" text.
/challenge
    Main command of the bot, it provides certain amount of challenges being up to 18 challenges.
    By pressing and sending any requested key being from a to d, you can respond the questions
    provided by the bot.
